# Filmoteca

Este es un ejemplo de proyecto colaborativo sobre el cual se
trabajará en el [ejercicio del Bloque X: Gestión de proyectos y Pull Request]

El proyecto consiste en la creación de una base de datos de películas,
series y libros. Cada participante aportará una o más entradas a la base de datos.

## Estructura del proyecto

El proyecto está organizado en tres directorios:

- `pelicules`: Contiene los datos de las películas.
- `series`: Contiene los datos de las series.
- `llibres`: Contiene los datos de los libros.

Cada elemento de la base de datos se encuentra en un fichero
de texto en formato Markdown dentro del directorio correspondiente.

## Contribuir al proyecto

Para contribuir al proyecto, hay que seguir los siguientes pasos:

1. Hacer un *fork* del proyecto.
2. Clonar el repositorio en tu dispositivo.
3. Crear una rama para tu contribución.
    - Hay que utilizar un prefijo en la rama dependiendo del tipo de contribución:
        - Películas: `pelicula/`
        - Series: `serie/`
        - Libros: `llibre/`

4. Añadir los datos de tu película, serie o libro en el directorio correspondiente.
5. Puedes realizar todos los *commits* que consideres necesarios.
6. Publicar la rama en tu repositorio.
7. Hacer una solicitud de *Pull Request* al repositorio original.
    - Hay que añadir un título y una descripción a la solicitud de incorporación.
